# 最大子数组和

## Version 1
這麼做應該沒錯，但是會超時，因為遍歷了兩次數組。
```go
func maxSubArray(nums []int) int {
	if len(nums) == 1 {
		return nums[0]
	}

	result := nums[0]
	for i := 0; i < len(nums); i++ {
		tmp := 0

		if result < nums[i] {
			result = nums[i]
		}

		for j := i; j < len(nums); j++ {
			tmp += nums[j]
			if result < tmp {
				result = tmp
			}
		}
	}

	return result
}
```
